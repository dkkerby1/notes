# Networking & Cybersecurity Glossary

## Networking
IP Address – unique identifier for a device
MAC Address – hardware identifier
DNS – translates domain names to IP addresses
TCP – reliable, connection-based protocol
UDP – fast, connectionless protocol
Port – logical endpoint for services
Firewall – controls allowed network traffic
NAT – translates private IPs to public IPs

## Security
Vulnerability – a weakness in a system
Threat – potential cause of harm
Risk – likelihood + impact of a threat
Exploit – method used to take advantage of a vulnerability
Patch – fix for a vulnerability
Zero-Day – unknown/unpatched vulnerability
Attack Surface – all possible entry points
Defense in Depth – layered security approach

## Web Security
HTTPS – encrypted HTTP
TLS – encryption protocol
Certificate – proves website identity
Session – user interaction period
Cookie – stored session data
